# 🌍 Travel Planner

A full-stack web application that helps users plan, organize, and manage their trips efficiently.
With an intuitive interface and powerful backend, users can explore destinations, create itineraries, and manage travel details all in one place.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Vinay019-code/Travel-planner.git
cd Travel-planner
```

### 2️⃣ Install Dependencies

```
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=8080
```

### 4️⃣ Run the Application

```
npm start
```

👉 Open in browser:

```
http://localhost:8080
```
